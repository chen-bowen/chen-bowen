## Featured project: Instacart Two-Tower Recommender

**From purchase history to ranked product recommendations on a 50K+ item catalog.**[cite:16][cite:20][cite:21][cite:22]

- Built a two-tower recommendation model using a distilled sentence-transformer over the Instacart dataset to generate user and item embeddings from purchase history.[cite:16][cite:20][cite:21][cite:22]  
- Trained on ~1.2M interactions under tight hardware constraints, reaching solid ranking quality (e.g., NDCG@10 ≈ 0.43 after 1 epoch) while keeping inference practical for real-world use.[cite:16][cite:20]  
- Designed the system with production in mind: containerized FastAPI service, clear interfaces for online serving, and monitoring/feedback hooks for continuous improvement.[cite:16][cite:20][cite:21][cite:22]  
- Wrote an accompanying blog series walking through training, serving, and deployment of the system end-to-end.[cite:21][cite:22]

👉 Repo: `chen-bowen/instacart_next_order_recommendation`  
👉 Model: published on Hugging Face Hub as `chenbowen184/instacart-two-tower-sbert`[cite:16][cite:21][cite:22]
